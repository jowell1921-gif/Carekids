# Carekids
Carekids is an app designed to provide information and monitor children with illnesses. Created for use in any hospital, it interactively teaches children about their treatment, improvement, and recovery.
Spanish: Carekids es una aplicación orientada a la información y seguimiento de niños con enfermedades. Creada para cualquier hospital, es una aplicacion que enseñá a los niños interactivamente su tratamiento, mejora y recuperacion.
